HTML

  • The title is located in the head element.
  • The head element contains information about the webpage.
  • The head element also contains metadata that is not visible on the webpage.
  • The body element represents the visible content shown to the user.
  • The footer often contains items such as author's name, emails, other contact information.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS=Cascading Style Sheets
  • Three types inline, internal, external
  • External stylesheet allows for keeping CSS rules in different file
  • Link CSS rules to HTML by using link element in head of HTML document

Git

  • All git commands in Gitbash are preceded by git
  • cd: change directory
  • Go to main: git checkout main
  • Branch check: git status
  • Create branch: git checkout -b feature/....
  • Add information: git add -A
  • Commit information: git commit -m "...."
  • Sync changes with Git Hub Repo: git push origin feature/.....
  • Make sure you have latest version: git pull origin main

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional statements and loops-Control the order in which code is executed.
  • An array is a group of related data stored in a single variable.
  • An index is unique number used to locate a data item in an array.
  • Arrays are 0 indexed therefore when counting 0 is the first numeral.
  • For loops contain three statements
  • /* *\use these symbols to comment code
  • A function is a set of instructions that tell a computer how to perform a task.
  • There are two requirements for Javascript functions
